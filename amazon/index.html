1FEATURES
⚡
TRANS-AM
保存済み直販URLで在庫待ち→在庫検出→今すぐ買う を全自動の高速ループ。
🏠
ホームアイコン起動
ショートカットのアイコン1タップで、その商品のTRANS-AMを自動発動。
🛡
直販オンリー / 二重防止
Amazon直販のみ対象。数量更新検出で二重購入を防止(トグル切替)。
📋
ログ & 通知
全動作をログ保存・CSV出力。Discordへ成功/失敗を通知(任意)。
2SETUP
1
App Store で 「Userscripts」(作者: Justin Wasack)を無料インストール。
2
iOS設定 → Safari → 拡張機能 → Userscripts を ON、すべてのサイトで許可。
3
上の 「⬇ インストール」 をタップ → Userscripts の Save / Install。
※ 旧 Netlify版が入っていたら Userscripts アプリで 削除(競合防止)。
4
Safari で amazon.co.jp の商品ページを開く → 右下にパネル(v0.3.8.99)が出れば成功。
5
パネル ⚙設定 で Discord webhook を入力(任意)。あとは 🛒 / ⚡TRANS-AM / 🏠 で運用開始。
💡設定・購入・通知 ── すべて UserScript 内で完結。Discord webhook はこの端末にのみ保存され、別ドメインへの送信や外部アカウント登録はありません。更新は @updateURL で自動。
