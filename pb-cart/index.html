1 FEATURES
⚡
高速カート投入
青ボタン検知 → 自動 POST。 form.submit 経由で人間操作と同等。
🛡
BAN リスク低減
混雑表示中の自動停止 / 発売後 30 分の警戒モードで連打を抑制。
🔔
Discord 通知
確保した瞬間、 商品名 / 注文 ID / カート画面リンクを通知。
📱
複数タブ並行
商品 A タブ / 商品 B タブを同時監視。 停止 / 再開はタブごと独立。
2 SETUP
1
App Store で 「UserScripts」 (作者: Justin Wasack) を無料インストール。
2
iOS設定 → Safari → 機能拡張 → UserScripts を ON、 すべてのサイトで許可 に。
3
「⬇ インストール」 をタップ → UserScripts の Save ダイアログで Save。
4
Safari で https://p-bandai.jp/ を開く → 起動バナーが出れば成功。
5
右下 「⚙ 設定」 ボタンから商品を追加（通知を使うなら Discord Webhook もここで入力）→ 自動監視 開始。
📖 詳しい使い方は説明書を参照 (PDF)
💡設定・カート投入・通知 ── すべて UserScript 内で完結します。 別ドメインへの設定送信や Cookie 共有、 外部アカウント登録は一切ありません。
